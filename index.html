Data Deletion Instructions (Callback Protocol)

In compliance with Meta Developer Guidelines, we provide a clear method for users to remove their profile connections from our applications.

Because our gateway operates completely stateless, we do not save any permanent records, user contact rosters, or historic chat metadata. As a result, there is no permanent data database to wipe clean.

If you want to disconnect your Meta profile or communication assets from our internal system paths entirely, please follow these steps:

  1. Go to your Facebook Profile settings and click on Apps and Websites.
  2. Locate the application named WB.
  3. Click the Remove button next to the application entry.
  4. To request immediate manual verification that your number is cleared from transient memory routers, send an email to: info@emanbakery.online.
